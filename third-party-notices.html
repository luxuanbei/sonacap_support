Current Bundled or Linked Components

  • WhisperKit / Argmax OSS Swift package, including its upstream notices.
  • OpenAI Whisper base model assets bundled with the app for local fallback speech recognition.
  • DeepFilterNet3 Core ML model assets bundled with the app for speech enhancement experiments and fallback processing.
  • Related bundled model files, tokenizer files, and converted Core ML model files required by those components.

Not Included in Current iOS Notices
